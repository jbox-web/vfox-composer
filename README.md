# vfox-composer

[![CI](https://github.com/jbox-web/vfox-composer/workflows/CI/badge.svg)](https://github.com/jbox-web/vfox-composer/actions)

A plugin for the [mise version manager](https://github.com/jdx/mise) to install [composer](https://getcomposer.org)

## Install

```
mise plugin install composer https://github.com/jbox-web/vfox-composer.git
```
